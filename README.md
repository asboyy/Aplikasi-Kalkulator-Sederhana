# Aplikasi Kalkulator Sederhana

Aplikasi Kalkulator Sederhana adalah perangkat lunak desktop ringan yang dirancang untuk melakukan operasi matematika dasar dengan cepat dan efisien. Fokus utama proyek ini adalah menciptakan antarmuka yang bersih dan alur logika yang akurat untuk penggunaan sehari-hari.

## 🚀 Fitur

- **Operasi Standar:** Mendukung penjumlahan, pengurangan, perkalian, dan pembagian.
- **Antarmuka Intuitif:** Tata letak tombol yang familiar dan mudah digunakan.
- **Presisi Perhitungan:** Menangani angka desimal dan operasi aritmatika dengan akurasi tinggi.
- **Tombol Pembersih (Clear):** Fitur untuk menghapus input atau mereset perhitungan secara instan.

## 🛠️ Teknologi yang Digunakan

- **Bahasa Pemrograman:** Visual Basic .NET (VB.NET)
- **Platform:** .NET Framework
- **IDE:** Visual Studio

## 📋 Prasyarat

Untuk menjalankan aplikasi ini di komputer Anda, pastikan Anda memiliki:
- Windows OS (7/8/10/11).
- .NET Framework Runtime yang sesuai.
- Visual Studio (jika ingin melihat atau memodifikasi kode sumber).

# Aplikasi Kalkulator Sederhana (Mobile Version) 📱

Aplikasi Kalkulator Sederhana adalah alat bantu hitung digital yang dirancang khusus untuk perangkat mobile. Fokus utama proyek ini adalah menghadirkan antarmuka yang responsif, ringan, dan mudah dioperasikan dengan satu tangan (one-handed use) untuk kebutuhan perhitungan aritmatika sehari-hari.

## 🚀 Fitur Utama

- **Operasi Aritmatika Dasar:** Mendukung penjumlahan, pengurangan, perkalian, dan pembagian secara akurat.
- **UI Responsif:** Tata letak tombol yang menyesuaikan dengan berbagai ukuran layar smartphone (Phone & Tablet).
- **History Calculation:** (Opsional) Fitur untuk melihat riwayat perhitungan sebelumnya dalam satu sesi.
- **Error Handling:** Validasi cerdas untuk mencegah kesalahan input seperti pembagian dengan angka nol.
- **Input Cepat:** Tombol numerik dan operator yang dirancang cukup besar untuk meminimalisir kesalahan tekan.

## 🛠️ Teknologi yang Digunakan

- **Platform:** Android / iOS (Silakan sesuaikan)
- **Bahasa Pemrograman:** Java / Kotlin / Dart (Flutter)
- **Environment:** Android Studio / VS Code
- **Layout:** XML / Jetpack Compose / Flutter Widgets

## 🧠 Konsep Pengembangan Mobile

Proyek ini mendemonstrasikan beberapa aspek teknis pengembangan mobile:
1. **Event Driven Programming:** Menangani interaksi sentuhan (touch events) pada setiap tombol.
2. **State Management:** Mengelola perubahan nilai pada layar secara dinamis saat tombol ditekan.
3. **Logic Handling:** Implementasi urutan operasi matematika (presedensi operator) yang tepat.
4. **Resources Management:** Penggunaan aset ikon dan warna yang efisien untuk menjaga ukuran file APK tetap kecil.
